Statistics

Here I'd like to explain the statistics that we offer. For example:
Block Behind - This tells the user if their node is up-to-date with the network. If the node has been offline for a while, the user will be able to monitor the node catching up to the network.
Hash Rate - This is the raw power of the network. Gorilla pulls this data from a 3rd party. Users can use this to compare to different coins' hash rate.

Backing Up

The only thing that you'll need to back up is your wallet.dat file. We'll be adding a menu item for a one-click backup process. Mini Computing recommends backing up to a separate flash drive. Step 1: Insert a flash drive into an open port on your Mini.
Step 2: Open the Terminal and type the following command.
sudo something -or other && something else
